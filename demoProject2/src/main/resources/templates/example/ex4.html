        th:if = ${조건식}
        - 조건식이 true인 경우에만
        해당 속성이 작성된 요소를 화면에 출력함

        th:unless = ${조건식}
        - 조건식이 false 경우에만
        해당 속성이 작성된 요소를 화면에 출력함

삼항 연산자

        - 타임리프 속성(th:)에
        삼항 연산자(조건식 ? 참 : 거짓) 작성 가능

삼항 연산자

Elvis 연산자 ( ?: )

        [작성법]
        값 ?: 값이 없을 때

        - 삼항 연산자에서
        조건식 자리에 값(변수명)만 작성
        (==, != 등의 연산자 사용 X)

        - 우변에는 값이 없을 때에 대한 값만 작성

        - 조건식 값이 존재하면 해당 값 출력
        없으면 우변을 출력
        
        -> 해당 값이 있는지 없는지에 따라 동작하는 연산자

No_Operation 연산자 ( 값 ? : _)

        - 조건식의 값이 없을 경우 (== null)
        타임리프 코드를 해석하지 않는 연산자

        - 타임리프 코드 해석 x
            -> 일반 HTML 태그로 동작
                -> HTML 태그 사이 내용(content)이 화면에 출력
